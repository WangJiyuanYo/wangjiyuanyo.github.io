# Hibernate框架

  1. 应用在javaee三层结构中Dao层框架

  2. 在Dao层对数据库CRUD操作，hibernate底层代码就是JDBC，Hibernate对JDBC进行封装，不需要复杂的代码，不需要写Sql语句实现；

  3. Hibernate开源轻量级框架

  4. Hibernate版本

    Hibernate3.X

    Hibernate4.X

# 什么是ORM思想

  1. Hiber 使用 ORM思想对数据库进行CRUD操作

  2. 在Web阶段学习Javabean 跟正确的叫法 实体类

  3. ORM：对象关系映射

    文字描述：

    （１）让实体类和数据库表进行一一对应

    ​ 让实体类首先和数据库表对应

    ​ 让实体属性和表里面字段对应

    （２）不需要直接操作数据库表，而操作表对应的实体类
