# Nginx-for-Windows

# 1.下载Nginx

Nginx (opens new window)

# 2.启动

运行Nginx.exe（nginx -c conf\nginx.conf）

# 3.使用

本地链接 (opens new window)

# 4.关闭

nginx -s stop 或taskkill /F /IM nginx.exe > nul


tasklist /fi "imagename eq nginx.exe" #查看进程

taskkill /pid xxxx -f #强制关闭

# 5.常用配置

C:\nginx\conf\nginx.conf,使用自己定义的conf文件如my.conf，命令为nginx -c conf\my.conf

nginx -s stop fast shutdown
nginx -s quit graceful shutdown
nginx -s reload changing configuration, starting new worker processes with a new configuration, graceful shutdown of old worker processes
nginx -s reopen re-opening log files
